# 📊 OKR 성과 관리 대시보드 (OKR Goal Dashboard)

## 1. 프로젝트 소개
**"경영학의 OKR 이론을 IT 기술로 구현하다"**

본 프로젝트는 경영학의 핵심 성과 관리 기법인 **OKR(Objectives and Key Results)** 방법론을 웹 애플리케이션으로 구현한 개인 맞춤형 성과 관리 대시보드입니다.

기존의 단순한 '할 일 목록(To-Do List)' 방식은 목표의 위계 구조와 마감 기한을 한눈에 파악하기 어렵다는 한계가 있었습니다. 이를 해결하기 위해 **상위 목표와 세부 실천 과제를 구조화**하고, **마감일(D-Day)과 주간 성과 추이를 시각화**하여 사용자가 주도적으로 목표를 달성하도록 돕는 도구를 제작했습니다.

## 2. 구현한 기능
본 프로젝트는 **사용자 경험(UX)**과 **데이터 시각화**에 중점을 두어 개발되었습니다.

* **스마트 목표 수립 (OKR System)**
    * 상위 목표(Objective) 하위에 여러 개의 핵심 결과(Key Results)를 추가할 수 있는 계층형 구조를 구현했습니다.
    * 복잡한 세부 항목은 '아코디언(펼치기/접기)' UI로 처리하여 깔끔한 대시보드 환경을 제공합니다.

* **마감 기한 및 일정 관리 (Time Management)**
    * 목표 설정 시 마감일을 입력하면, 현재 날짜를 기준으로 남은 일수(D-Day)를 자동 계산합니다.
    * 상태에 따라 `D-7`, `Today`, `Overdue(지연)` 등의 배지(Badge)가 자동으로 부착되어 업무 우선순위를 파악할 수 있습니다.

* **데이터 시각화 및 피드백 (Analytics)**
    * **주간 성과 차트:** 외부 라이브러리 없이 JavaScript 알고리즘으로 최근 7일간의 달성률 데이터를 집계하여 막대그래프로 시각화했습니다.
    * **동적 테마:** 달성률이 70% 이상일 경우 테마 색상이 주황색에서 보라색으로 변경되어 시각적인 보상을 제공합니다.

* **데이터 영구 저장 (LocalStorage)**
    * 브라우저의 `localStorage` API를 활용하여, 별도의 로그인이나 서버 없이도 입력한 데이터가 영구적으로 보존됩니다.

* **반응형 웹 디자인 (Responsive Design)**
    * 사이드바가 포함된 전문적인 대시보드 레이아웃을 기본으로 하며, 모바일 환경에서는 최적화된 리스트 형태로 변환됩니다.

## 3. 실행 방법
별도의 서버 설치 과정 없이, 웹 브라우저만 있으면 즉시 실행 가능합니다.

1.  GitHub 저장소에서 프로젝트를 다운로드(`ZIP`)하거나 클론(`git clone`)합니다.
2.  폴더 내의 **`index.html`** 파일을 더블 클릭하여 실행합니다.
3.  목표 입력창에 목표와 마감일을 입력하고 `목표 추가` 버튼을 눌러 테스트합니다.

## 4. 사용한 기술스택
* **프론트엔드(Frontend):** HTML5, CSS3
* **스크립트(Script):** JavaScript (Vanilla ES6+)
* **데이터 저장(Database):** Web LocalStorage API (Client-side)
* **개발 도구(Tools):** Cursor AI, Visual Studio Code, Git

## 5. 기타 참고사항
* **브라우저 권장:** 최신 CSS Grid 및 ES6 문법이 사용되었으므로 **Google Chrome** 또는 **Microsoft Edge** 최신 버전 사용을 권장합니다. (Internet Explorer는 지원하지 않습니다.)
* **데이터 보안:** 모든 데이터는 사용자의 브라우저 내부에만 저장되므로, 개인정보 유출 걱정 없이 안전하게 사용할 수 있습니다. 단, 브라우저 캐시를 삭제하면 데이터가 초기화될 수 있습니다.
* **알림 권한:** 별도의 브라우저 알림 권한은 요구하지 않으나, 마감일이 지난 항목은 시각적으로 붉은색 배지로 경고 표시됩니다.
